## TAX System
## Prerequisites

* Java 1.8
* Maven

## Get the sources

1. git clone
2. cd TaxSystem

## Running Tests

3. mvn test

## Running

3. mvn package
4. cd target
5. java -cp TaxSystem-1.0-SNAPSHOT.jar com.challenge.InvoiceApp
